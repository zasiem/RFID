-File dosen berisikan data dosen, isikan data dosen sesuai format file yang ditentukan
-File log merupakan file untuk menyimpan log tapping, tidak perlu diedit edit karena datanya dinamis tergantung anda melakukan tapping
- loglecturer berisikan no_ruangan, nip, indikator. isikan terlebih dahulu nip dosen yang bisa tapping di nomor ruangan tersebut. jika indikatornya 0 maka belum tapping, jika 1 maka sudah tapping dosennya
- mahasiswa adalah file dari mahasiswa
- schedules adalah jadwal yang bisa dilakukan tapping, diisikan terlebih dahulu

DATA YANG WAJIB DI EDIT :
mahasiswa, schedules, loglecturer, dosen
